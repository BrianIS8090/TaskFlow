# Changelog

## [1.2.0] - 2026-02-04

### Добавлено
- Функция переноса задач на день назад (кнопка со стрелкой влево ←).
- Кнопка "Перенести на вчера" в desktop-версии с фиолетовым акцентом.
- Пункт меню "Перенести на вчера" в мобильном бургер-меню.
- Автоматическое уменьшение счётчика `postponeCount` при переносе задачи назад.

## [1.1.1] - 2026-02-04

### Исправлено
- Восстановлена конфигурация Firebase (`.env`) для корректной работы авторизации и БД.
- Исправлено отображение кнопок действий (редактировать/удалить) в списке чекпоинтов на мобильных устройствах: теперь они всегда видны, так как `hover` недоступен.

## [1.1.0] - 2026-02-03

### Добавлено
- Поддержка светлой темы (Light Mode) во всём приложении.
- Переключатель темы (Солнце/Луна) в сайдбаре слева от иконки пользователя.
- Полноценная нативная авторизация Google для Desktop (Tauri) через `tauri-plugin-oauth`.
- Локальный сервер авторизации (порт 14205) для перехвата токенов в Desktop версии.
- Подробные инструкции по разработке, сборке и деплою в [README.md](./README.md).

### Исправлено
- Ошибка "Command plugin:oauth|start not allowed by ACL" в конфигурации Tauri.
- Обработка URL редиректа: теперь приложение корректно слушает ответ через `onUrl`.
- Читаемость текста в светлой теме: все компоненты (`TaskItem`, `SearchModal`, `ConfirmDialog`) адаптированы под светлый фон.
- Настройка переменных среды (PATH) для корректной работы компилятора Rust.

### Инфраструктура
- Выполнен деплой веб-версии на Firebase Hosting.
- Обновлены зависимости Tauri (OAuth плагин и JS-клиент).

## [1.0.0] - 2026-02-03

### Добавлено
- Глобальный поиск задач с модальным окном
- Навигация по датам (< >) в шапке
- Мобильное бургер-меню для действий задачи
- Подтверждение удаления задач и чекпоинтов
- Отображение версии приложения в Sidebar
- Счётчик переносов задачи (`postponeCount`)
- Дата завершения задачи (`completedAt`)

### Исправлено
- Модальные окна теперь отображаются поверх всего интерфейса (React Portal)
- Бургер-меню выносится за пределы карточки задачи
- Поиск перемещён в Sidebar под календарь
- Дата в desktop версии центрирована с увеличенными отступами
- Выравнивание чекбокса чекпоинта по началу строки
- Перенос длинного текста в чекпоинтах

### Удалено
- Mock-данные задач
